Encoding Task

You will perform an encoding task. Specifically, you will be given sequences of three random capital letters. We call the three letters combination a “Word” for simplicity.

You have to allocate a number to each capital letter. The encoding code can be found in the table below the corresponding letter. For each Word, the encoding table changes.

For illustration, consider the following screenshot


In this example, the three capital letters: “E”, “T” and “K” have to be encoded. The solution follows immediately from the encoding table:

  • The encryption code for “E” is 627 (already entered by the participant)
  • The encryption code for “T” is 784 (already entered by the participant)
  • The encryption code for “K” is 333 (not entered yet by the participant)

You will have {{ minutes_task|to0 }} minutes to encode as many words into numbers as possible. Note that each letter must be encoded correctly to continue to the next Word. You will be paid {{ Constants.piece_rate }} for each correct Word.

After completing all the letters of a Word, click on submit. If the three letter codes are correct, a new Word and a new encoding table will appear. Otherwise, you will see an alert informing you that you have some mistakes in your encoding.

You have to check and correct the codes until the Word is correct.

Each new encoding table is randomly generated in two steps:

  1. The computer program randomly selects in the table a new set of three-digit numbers to be used for the capital letters encoding.
  2. Additionally, the computer program shuffles the position of the capital letters in the table. Please note that the program always uses all 26 capital letters of the alphabet.
